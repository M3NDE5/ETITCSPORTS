Diseña la interfaz completa de una plataforma web para la gestión y monitoreo de torneos de fútbol en una institución educativa llamada Escuela Tecnológica Instituto Técnico Central.

El diseño debe ser moderno, limpio y fácil de usar, pensado para estudiantes, organizadores y administradores del torneo.

Estilo visual

Usar tonos verdes suaves como color principal, relacionados con el fútbol y el campo de juego.

Combinar con blanco y gris claro para mantener un diseño limpio.

Estilo minimalista, moderno y educativo.

Usar iconografía deportiva (balón, trofeo, campo, jugadores).

Componentes tipo dashboard con tarjetas, tablas y gráficos.
